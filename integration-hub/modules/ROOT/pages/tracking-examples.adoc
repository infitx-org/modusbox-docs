= Tracking Examples
:keywords: Anypoint b2b Integration Hub concepts

PortX Transaction Monitor enables you to monitor B2B transaction data traveling through Integration Hub.
This section identifies examples in which Integration Hub (IHub) is useful to track B2B exchanges. A basic knowledge of Ihub is assumed; see xref:index.adoc[Integration Hub] for more information.

Transaction Monitor allows you to track events. An event is defined as the type of data that travels through Ihub. Track events under the following tracking categories.

* <<Tracking Transactions>> 
* <<Tracking Transmissions>>
* <<Tracking Errors>>
* <<Tracking Documents>>

For information on replaying a transaction see <<Replaying an Incomplete Transaction>>. 


This page identifies components of the Transaction Monitoring Page; see also xref:central-pane-elements.adoc[Central Pane Elements] for more information.
